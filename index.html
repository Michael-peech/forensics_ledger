System User Guide

A. Log Initial Evidence

Used strictly at the crime scene. Fills both Case Name and unique Item ID. Accepts any file format (PDF, JPG, RAW disk images). You become the Custodian.

B. Transfer Custody

To transfer, you must input the exact Item ID. **Security Feature:** You must also paste the correct Walrus ID. If you paste the wrong ID, the system blocks the transfer to prevent evidence tampering.

D. Evidence Locker & Lab Reports

Shows items you hold. If you are a Lab Tech (i.e., the item was transferred to you using the "Checked out for Data Analysis" action), the system will unlock the restricted Analysis and Evaluation writing scripts for you.
