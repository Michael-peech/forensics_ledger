Protocol Documentation

A. Intake Protocol

Initiates a new asset on the EVM registry. A case name and unique identifier are mapped to the decentralized storage hash. Your wallet is mathematically locked as the origin node.

B. Transfer & Verification

To execute a handoff, input the Case Name (or Item ID). The system will attempt a reverse-lookup. Integrity Check: You must supply the correct Walrus Hash. If the payload does not match the registry, the transaction reverts.

D. Conditional Lab Access

Accessing an asset in your locker queries its state. If the last recorded action is Forensic Imaging or Data Analysis, the system grants write-access to the Analysis and Evaluation matrices.
